Beekeeper Studio is a cross-platform desktop application that gives you a visual interface for working with databases. Whether you're browsing table data, writing SQL queries, or managing schema, Beekeeper Studio is designed to make your work faster and more enjoyable than using command-line tools like `psql` or `mysql`.

## What Beekeeper Studio does

Beekeeper Studio replaces the need to memorize CLI commands with a clean, tabbed GUI. You can connect to a database, explore its tables, write and save queries, edit rows directly, and export data — all without leaving the app.

It runs natively on macOS, Windows, and Linux with **full feature parity across all platforms**, so you get the same experience regardless of your operating system.

## Supported platforms

Beekeeper Studio runs on macOS (Intel and Apple Silicon), Windows (x86_64), and Linux (x86_64 and ARM64). There are no plans for iOS or Android versions.
