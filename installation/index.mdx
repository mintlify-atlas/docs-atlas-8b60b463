Beekeeper Studio is a desktop application available for macOS, Windows, and Linux. You can download the community edition for free — no account or credit card required — from the [Beekeeper Studio download page](https://beekeeperstudio.io/get-community). This page covers installation options across all supported platforms.

<Note>
Beekeeper Studio provides **full feature parity** across all operating systems. You get the same interface and capabilities whether you're on macOS, Windows, or Linux.
</Note>

## Choose your platform

<Columns cols={3}>
  <Card title="macOS" icon="apple" href="/installation/macos">
    Install via `.dmg` on Intel or Apple Silicon Macs. Supports both architectures natively.
  </Card>
  <Card title="Windows" icon="windows" href="/installation/windows">
    Run the standard `.exe` installer or use the portable version — no admin rights needed.
  </Card>
  <Card title="Linux" icon="linux" href="/installation/linux">
    Install via AppImage, DEB, RPM, Snap, Flatpak, or Arch packages. Supports x86_64 and ARM64.
  </Card>
</Columns>

## Supported architectures

| Platform | Architectures |
|----------|--------------|
| macOS    | Intel (x86_64), Apple Silicon (ARM64) |
| Windows  | x86_64 |
| Linux    | x86_64, ARM64 |

<Tip>
On Linux, we recommend the AppImage for the most consistent experience across all distributions. It runs on almost every Linux distro without requiring root access.
</Tip>
