---
title: "Install Beekeeper Studio on Windows"
description: "How to install Beekeeper Studio on Windows using the standard installer or the portable version that requires no admin privileges."
---

Beekeeper Studio on Windows comes in two flavors: a standard installer that sets up the app like any other Windows program, and a portable version you can run from any folder without installing anything. Both are available free from the Beekeeper Studio website.

<Note>
Beekeeper Studio for Windows currently supports **x86_64** (64-bit) architecture only. ARM64 support is planned for a future release.
</Note>

## Standard installer

<Steps>
  <Step title="Download the installer">
    Go to the [Beekeeper Studio download page](https://beekeeperstudio.io/get-community) and download the Windows `.exe` installer.
  </Step>

  <Step title="Run the installer">
    Double-click the downloaded `.exe` file and follow the on-screen prompts. Beekeeper Studio will be installed and a shortcut will be added to your Start menu.
  </Step>

  <Step title="Launch Beekeeper Studio">
    Open Beekeeper Studio from the Start menu or from the desktop shortcut created during installation.
  </Step>
</Steps>

## Portable version

The portable version of Beekeeper Studio does not require installation and does not write to the Windows registry. This is useful if you:

- Don't have administrator rights on the machine
- Want to run Beekeeper Studio from a USB drive or shared folder
- Prefer not to modify your system

Download the portable version from the [Beekeeper Studio download page](https://beekeeperstudio.io/get-community), extract the archive, and run `beekeeper-studio.exe` directly.
