---

Beekeeper Studio runs natively on both Intel and Apple Silicon Macs. This page covers how to download and install Beekeeper Studio on macOS, how to choose the right installer for your architecture, and a known limitation for Oracle Database users on Apple Silicon.

## Download and install

<Steps>
  <Step title="Download the installer">
    Go to the [Beekeeper Studio download page](https://beekeeperstudio.io/get-community) and download the `.dmg` file for macOS.

    Make sure you select the correct version for your Mac:

    - **Apple Silicon (M1, M2, M3, M4)**: choose the ARM64 / Apple Silicon build
    - **Intel**: choose the Intel / x86_64 build

    If you're unsure which chip your Mac has, go to **Apple menu → About This Mac**.
  </Step>

  <Step title="Open the DMG">
    Double-click the downloaded `.dmg` file to mount it. A Finder window will open showing the Beekeeper Studio app.
  </Step>

  <Step title="Drag to Applications">
    Drag the Beekeeper Studio icon into the **Applications** folder shortcut shown in the same window.
  </Step>

  <Step title="Allow the app to run">
    By default, macOS prevents apps that weren't downloaded from the Mac App Store from opening. To allow Beekeeper Studio:

    1. Go to **System Settings → Privacy & Security**
    2. Scroll to the Security section
    3. Under "Allow applications downloaded from", select **App Store and identified developers**

    If macOS blocked the app when you first tried to open it, you'll also see an **Open Anyway** button here.
  </Step>

  <Step title="Launch Beekeeper Studio">
    Open Beekeeper Studio from your Applications folder or from Launchpad. The first time you open it, macOS may ask you to confirm you want to run it — click **Open**.
  </Step>
</Steps>

## Intel vs. Apple Silicon

Beekeeper Studio ships separate installers for each architecture. Download the one that matches your Mac's chip for the best performance.

<Columns cols={2}>
  <Card title="Apple Silicon (M1/M2/M3/M4)" icon="microchip">
    Download the ARM64 build for native performance on any Mac with an Apple Silicon chip.
  </Card>
  <Card title="Intel" icon="microchip">
    Download the x86_64 build for Intel-based Macs. This build also runs on Apple Silicon via Rosetta 2.
  </Card>
</Columns>

<Warning>
**Oracle Database users on Apple Silicon:** Oracle's database client libraries do not currently support Apple Silicon. If you need to connect to an Oracle Database, you must install the **Intel version** of Beekeeper Studio and [run it under Rosetta 2](https://support.apple.com/en-us/HT211861).
</Warning>
