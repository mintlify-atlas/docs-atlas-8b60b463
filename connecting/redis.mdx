---

Beekeeper Studio supports Redis as a first-class connection type, letting you browse keys, edit values, run commands through the AI shell, and view JSON documents when the ReJSON module is installed.

<Note>
  Redis is available in both the free **Community** edition and paid **Ultimate** editions of Beekeeper Studio.
</Note>

## Connection details

| Field | Default | Description |
|---|---|---|
| Host | `127.0.0.1` | IP address or hostname of your Redis server |
| Port | `6379` | Port your Redis server listens on |
| Username | — | Redis username (optional; Redis 6+ with ACLs) |
| Password | — | Redis password (optional) |

## Connect from the app

<Steps>
  <Step title="Open the connection screen">
    Launch Beekeeper Studio and open the connection screen.
  </Step>
  <Step title="Select Redis">
    Choose **Redis** from the connection type dropdown.
  </Step>
  <Step title="Enter connection details">
    Fill in the **host** and **port** fields. If your Redis server requires authentication, enter your **username** and/or **password**.
  </Step>
  <Step title="Configure SSL (optional)">
    If your server uses TLS, expand the **TLS/SSL** settings and configure as needed.
  </Step>
  <Step title="Connect">
    Click **Connect**.
  </Step>
</Steps>

## Username authentication (ACLs)

Redis 6 introduced [Access Control Lists (ACLs)](https://redis.io/docs/management/security/acl/), which let you create multiple users with individual permissions. If your server uses ACLs, enter your **username** along with your **password**.

For Redis servers older than version 6 that use only `requirepass` authentication, leave the username field blank and enter just your password.

## ReJSON support

If your Redis server has the [ReJSON module](https://redis.io/docs/stack/json/) installed, Beekeeper Studio renders JSON values stored with `JSON.SET` with syntax highlighting in the value viewer. You can edit JSON documents in place without writing raw Redis commands.

<Tip>
  Redis Stack bundles ReJSON (and other modules) by default, so ReJSON support is available automatically if you're running Redis Stack.
</Tip>

## SSH tunneling

To connect to a Redis server on a private network, use Beekeeper Studio's SSH tunnel feature. See the [SSH tunneling section in the connection overview](/connecting/overview#ssh-tunneling) for full setup instructions.
