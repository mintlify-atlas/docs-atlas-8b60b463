description: "Browse, filter, and edit MongoDB collections using a table view, SQL queries, or a REPL shell. Requires the Ultimate edition. Early implementation."
---

Beekeeper Studio supports MongoDB, giving you a familiar table-based interface to browse collections, sort and filter data, edit documents, and run queries — either using SQL or directly against the MongoDB shell. MongoDB support is in active development, with new features added regularly.

<Note>
  MongoDB requires the **Ultimate** (paid) edition of Beekeeper Studio.
</Note>

<Warning>
  MongoDB support is an early implementation. Core features work, but some capabilities are still in progress. Check back frequently for updates.
</Warning>

## Connect from the app

To connect to MongoDB, select **MongoDB** from the connection type dropdown and enter your connection string or individual connection fields (host, port, username, password, and database). Click **Connect** when ready.

<Tip>
  MongoDB connection strings follow the format `mongodb://username:password@host:27017/database`. You can paste one into the URL import box at the top of the connection form.
</Tip>
